# Computação Gráfica e Geométrica

Este repositório contém material de estudo sobre computação gráfica e geométrica, incluindo teoria, exercícios e códigos. O material foi criado para ajudar estudantes e profissionais a aprender sobre computação gráfica e geométrica de uma forma prática e acessível.

O repositório está dividido em várias seções, cada uma com foco em um tópico diferente de computação gráfica e geométrica. A seção Teoria contém notas de aula e artigos sobre os fundamentos de computação gráfica e geométrica. A seção Exercícios contém exercícios para ajudar os estudantes a aplicar os conceitos aprendidos na seção Teoria. A seção Códigos contém implementações de algoritmos de computação gráfica e geométrica em diferentes linguagens de programação.

O repositório é mantido atualizado com novos recursos e atualizações. Os usuários são incentivados a contribuir com o repositório adicionando novas informações, exercícios e códigos.
